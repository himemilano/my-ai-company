地質調査におけるダイヤモンドビットの選定と、インプリグネーテッドビット特有のメカニズムについて、資料に基づき詳細に解説します。
1. ダイヤモンドビットの主要2形式と比較
ダイヤモンドビットは、刃先の構造によって大きく**「サーフェスセット型」と「インプリグネーテッド型」**に分類されます
。
地質別選定基準・比較表
岩盤の強度（一軸圧縮強度）や状態に応じて、以下のように使い分けます
。
項目
サーフェスセット型（表面植付け）
インプリグネーテッド型（粉末焼結）
構造
比較的大粒のダイヤをマトリックス表面に植え込んだもの
。
ダイヤ粉末をマトリックス金属と混合して焼結したもの
。
対象岩盤
軟岩 〜 硬岩
。
硬岩 〜 極硬岩、破砕帯、礫層
。
ダイヤ粒度
岩盤が硬くなるほど小粒にする（1/8ct 〜 1/60ct）
。
極微粒のパウダー状（30 〜 60メッシュ）を使用
。
マトリックス
岩盤が硬くなるほど軟らかいものを使用し、ダイヤの突出を保つ
。
岩盤強度に合わせ、ダイヤの摩耗と同期して摩耗する硬さを選定
。
切削の性質
ダイヤの「エッジ」で直接削る。高速回転が必要
。
マトリックスが摩耗しながら新しい刃を出す「自生作用」で削る
。
破砕帯への適応
衝撃に弱いため、不向きとされる
。
耐衝撃性に優れ、破砕帯やコア詰まりしやすい地層に適する
。
2. インプリグネーテッドビットの「自生作用」メカニズム
試験において記述や4択問題で問われやすい、自生作用（セルフシャープニング作用）の仕組みを詳細に解説します。
メカニズムの工程
粉末の分散: ビットの切刃部分（鋳込層）の全体に、ダイヤモンドの粉末が均一に分散・配置されています
。
マトリックスの研削: 掘進中、ダイヤモンドが岩石を削ると同時に、それを保持している金属（マトリックス）も岩石粉（スライム）によって少しずつ摩耗（研削）していきます
。
新しい切刃の露出: 表面に出ていたダイヤモンドが摩耗して丸くなったり脱落したりすると、周囲のマトリックスが削られることで、その下層に埋まっていた新しいダイヤモンドが次々と表面に現れます
。
切削力の維持: このプロセスが繰り返されることにより、ビットを研磨し直す（リセットする）手間をかけず、鋳込層がなくなるまで鋭い切れ味を維持したまま掘進を続けることができます
。
試験対策上の重要ルール
マトリックス硬さの選定: 自生作用を正常に働かせるには、地質に応じたマトリックスの「硬さ」が重要です。
マトリックスが硬すぎる：金属が摩耗せず、新しいダイヤが出てこないため、切れ味が悪くなる（目詰まり状態）
。
マトリックスが軟らかすぎる：金属が早く摩耗しすぎ、ダイヤが仕事をする前に脱落してしまい、ビット寿命が極端に短くなる
。
現場での対処: もし自生作用がうまく働かずダイヤの露出が悪くなった場合は、サンドブラスト（研磨）を行うか、よりマトリックスの軟らかいビットに交換する必要があります
。
3. 試験で狙われる「ひっかけ」ポイント
回転数の誤解: ダイヤモンドビット（特にインプリ型）は、メタルビットとは異なり、一般に高速回転で使用することでその能力を発揮します
。
適応地質の逆転: 「極硬岩には大粒のダイヤを植えたサーフェスビットが最適である」という設問は誤りです。正しくは、極硬岩には微粒ダイヤのインプリグネーテッドビットが適しています
。
リセットの可否: メタルビットは現場でチップの植え替え（リセット）が可能ですが、ダイヤモンドビットは現場でのリセットができず、多くの予備品が必要になります
