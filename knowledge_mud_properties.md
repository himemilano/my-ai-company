ボーリング調査において、泥水（でいすい）の管理は孔内事故を防ぎ、精度の高いデータを取得するために極めて重要です。資料に基づき、泥水の性質、管理基準、および実務・試験上の重要ポイントを詳細に整理します。
1. 泥水の主な役割
泥水は単なる循環水ではなく、以下の重要な工学的役割を担っています
。
スライムの排除: 掘り屑を地表へ運び出し、孔底を清掃する
。
孔壁の保護（泥壁形成）: 孔壁に薄くて強い「泥壁（マッドケーキ）」を作り、地層の崩壊を防ぐ
。
ビットの冷却と潤滑: 摩擦熱を奪い、ロッドの回転抵抗を減少させる
。
抑留（ジャミング）の防止: 循環停止時にスライムが急激に沈殿して、ロッドやコアバーレルが動かなくなるのを防ぐ
。
湧水・ガスの抑制: 泥水の重さ（泥水圧）によって、地層からの水やガスの突出を抑える
。
2. 泥水の性質：定義と管理基準
試験で問われる主要な性質の定義と、一般的な管理手法について解説します。
① 比重（密度）
定義: 泥水の重さを表す指標。湧水圧や孔壁の側圧とバランスを取るために重要である
。
管理: 「マッドバランス」という専用の比重計で測定する
。
基準値: ベントナイト泥水の場合、一般的に 1.03〜1.10 程度とされる
。
② ファンネル粘性（粘性）
定義: 泥水の「ねばりけ」。スライムの運搬能力や、停止時の保持能力に関係する
。
管理: 漏斗（ファンネル）から一定量の泥水が流れ落ちる時間を計測する（500cc/500cc法などが一般的）
。
基準値: ベントナイト泥水では 25〜35秒 程度が標準である
。
③ ろ過量（脱水量）と 泥壁の厚さ（ケーキ厚）
定義:
ろ過量: 地層の中へ泥水の水分がどれだけ逃げてしまうかを示す
。
泥壁の厚さ: ろ過された後に孔壁に残る粘土の膜の厚さ
。
管理: 泥壁試験器を用い、7kgf/cm²の圧力を30分間かけて測定する
。
重要性: ろ過量が多く泥壁が厚いと、孔径が縮小してロッドが動かなくなったり、コア採取率が低下したりする
。
基準値: ベントナイト泥水では、ろ過量 7cc以下、泥壁 1mm以下 が良好とされる
。
④ 砂分（さぶん）
定義: 泥水の中に混じっている砂（スライム）の含有率
。
管理: 100〜200メッシュのふるいを用いて測定する
。
基準値: 5%以下 を目安とする
。砂分が増えると泥壁が厚くなり、孔内事故（抑留）の原因となる
。
⑤ pH（水素イオン指数）
定義: 泥水のアルカリ性・酸性の度合い
。
管理: pH試験紙やガラス電極pHメーターで測定する
。
重要性: セメントなどの混入によりアルカリ性に偏りすぎると、粘性が破壊され泥水としての機能が失われる
。

4. トラブル対策・管理の重要ポイント（試験頻出）
① 性質の変化と劣化対策
泥水は地層中の粘土やカルシウム（石灰岩、セメント等）が混入することで、常に性質が変化する
。
対策: 少なくとも 1日1回は泥水試験 を行い、性質を適正に維持すること
。
悪い泥水（ろ過量が多く泥壁が厚い）は、崩壊や孔内事故の直接的な原因となる
。
② 逸水（いっすい）対策
砂礫層や空洞のある地層で、泥水が地層に吸い込まれて戻ってこない現象
。
対策（逸水防止剤）: 「テルストップ（綿実）」、「テルシール（蛭石）」、「マッドシール（パルプ加工品）」などを混合して注入する
。
③ セメント混入への注意
泥水の中にセメントが混じるとpHが急上昇し、泥壁形成能力が著しく低下する
。
対策: 泥水使用中のセメンティング（グラウティング）は極力控え、実施した場合は孔内を十分に洗浄するか、全泥水を交換することが望ましい
。
④ 砂分の管理
現場での孔内事故の多くは、スライム（砂分）が増加して泥壁が肥厚することに起因する
。
対策: 沈殿バック（泥水バック）やデッチライン（樋）を適切に管理し、滞留時間を長くしてスライムを十分に分離させること
。
これらの数値データと管理理論を把握しておくことは、現場技術部門の設問において非常に重要です
。
