1. 固定ピストン式サンプラーとデニソン型サンプラーの徹底比較
両者は「土の乱れを最小限に抑える」という目的は共通していますが、採取のメカニズムと適応する土質が大きく異なります。
比較項目
固定ピストン式サンプラー
デニソン型サンプラー
物理的構造
サンプリング管の内部にピストンを装備。採取時にピストンを固定し、管を押し込む
。
スイベル式ダブルチューブの一種。**内管がビットよりも数cm先行（突出）**している
。
採取の原理
ピストンによって孔内を真空状態に保ち、土試料を吸い込むようにして管内に入れる
。
内管を地盤に突き刺しながら、その周囲を外管のビットで切削して採取する
。
対象地層
軟弱な粘性土（シルト、粘土）
。
硬質な粘性土、砂質土、崩れやすい地層
。
試料の品質
極めて高い（乱れが最も少ない高品質な試料が得られる）
。
高い（循環水による洗掘から試料が保護される）
。
主な用途
圧密試験や強度試験用の精密な試料採取
。
固定ピストン式では貫入が困難な、やや硬い地盤の試料採取
。
2. 構造の違いとメカニズムの詳細
固定ピストン式サンプラー
サンプリング管を地中に押し込む際、内部のピストンを一定の高さに固定しておくのが特徴です。これにより、管内に土が入る際に上部が真空に近い状態になり、土の膨張や圧縮を防いで、**地中にあるそのままの状態（不攪乱）**で試料を回収できます
。
デニソン型サンプラー
二重管構造（ダブルチューブ）になっており、内管が回転する外管よりも前方に突き出しています。
役割: 循環水（泥水）が内管の先端を通らずに外側に逃げるよう設計されているため、水流によってコアが洗掘（流失）されるのを防ぐことができます
。
調整: 土質の硬さに応じて、内管の突出量を調整して使用します
。
3. 現場での注意点
孔底の清掃（スライム処理）: サンプリングを開始する前に、孔底に溜まったスライム（掘り屑）を完全に除去しておくことが不可欠です。スライムが残っていると、純粋な地層試料ではなく「掘り屑の塊」を採取することになり、試験データが無効になります
。
押し込みスピードと圧力: 固定ピストン式では、急激な押し込みは試料の乱れを招くため、静的かつ一定の速度で貫入させる必要があります
。
泥水管理: 特に軟弱地盤や砂層では、サンプラーを引き抜く際の負圧で孔壁が崩壊しないよう、適切な比重の泥水で孔内を保護しておくことが重要です
。
4. 試験対策：土壌による見極めポイント
試験で「〇〇な土壌にはどちらが適するか」と問われた際の鉄板の判断基準です。
**「軟弱な粘性土」「極めて高品質」「真空吸引」**というキーワードが出たら？ → 正解は 「固定ピストン式サンプラー」 です
。
**「硬質な粘土」「砂層」「内管がビットより先行（突出）」「二重管式」**というキーワードが出たら？ → 正解は 「デニソン型サンプラー」 です
。
**「循環水による洗掘を防ぎたい」**という目的が強調されたら？ → 二重管でコアを守る構造を持つ 「デニソン型」 を選びます
。
この使い分けの根拠（真空で吸うか、二重管で守るか）を理解しておけば、応用問題にも自信を持って回答できます。
