提供された資料の「図2.5 ボーリング装置の全体図」
に基づき、装置の構成パーツ、その物理的役割、および試験で重要となる連動ロジックについて詳細に解説します。
この図は、地上にある動力・操縦系、水を循環させる送水系、ロッドを支える昇降系、そして地中にある掘削ツール系の4つのシステムが組み合わさった構造をしています。
1. 地上施設と動力・昇降システム（やぐら周辺）
三脚パイプやぐら
役割: 掘削装置全体を覆い、ロッドやケーシングの昇降（抜き差し）を支えるためのタワー状の構造物
。
試験のポイント: 耐荷重（ロッド総重量の3〜4倍以上）や設置の安定性が問われます
。
スナッチブロック（滑車）
役割: やぐらの頂部に吊り下げられ、巻上機からのワイヤロープを通す滑車
。
巻上機（ホイスト）
役割: ワイヤロープをドラムに巻き取ったり解いたりすることで、地中のロッドや器具類を昇降させる
。
試験のポイント: 瞬時の昇降切換えや制動（ブレーキ）が重要で、遊星歯車機構などが採用されます
。
原動機（エンジン/モータ）
役割: 装置全体の動力源
。電気、ガソリン、ディーゼルなどがあります
。
伝動装置・変速装置・操縦装置
役割: 原動機の動力をベルトやチェーンで各部（スイベルヘッドや巻上機）に伝え、回転数やトルクを調整する
。
重要ロジック: 地質に合わせて「ビットを高速回転させるか」「荷重をかけるか」をオペレーターがレバーで操作する心臓部です
。
2. 回転・給進システム（せん孔装置部）
スイベルヘッド（せん孔装置）
役割: ボーリングロッドに**「回転運動」と「進退運動（給進）」**を同時に与える装置
。
物理的構造: 中空のスピンドルの中をロッドが貫通し、チャックで固定されています
。
ロッドホルダ
役割: ロッドを継ぎ足す際や引き上げる際、ロッドが自重で穴の中に落下しないよう、孔口で一時的に保持（クランプ）する器具
。
試験のポイント: 落下事故防止のための最重要保安器具の一つです
。
3. 送水・循環システム（スライム排除系）
ボーリングポンプ
役割: 用水（清水や泥水）を高い圧力で送り出す
。
試験のポイント: ビットの冷却と、切削した粉（スライム）を地上へ排出するのが主な目的です
。
ウォータスイベル
役割: 固定されたポンプ側のホースと、回転しているロッドの頭部を接続し、回転を止めずに水を送り込むための特殊な継手
。
サクションホース・フートバルブ
役割: ポンプが泥水バックから水を吸い上げるための経路。フートバルブは逆流を防ぎ、ゴミの混入を避けます
。
泥水バック（沈殿槽）
役割: 孔内から戻ってきた水を受け、スライムを沈殿させてから再び綺麗な水をポンプで吸い上げるための循環槽
。
4. 地中掘削ツールシステム（孔内）
ドライブパイプ・ケーシングパイプ
役割: 穴の入り口付近や、崩れやすい地層を保護するために挿入する外管
。
ボーリングロッド
役割: 地上の回転と推力をビットに伝え、同時に水の通り道（導水管）となる中空の鋼管
。
セジメントチューブ
役割: コアバーレルのすぐ上に接続され、循環水で排出されきれなかった重いスライムを堆積させて回収するパイプ
。
コアバーレル
役割: 地中の土や岩のサンプル（コア）を壊さずに採取するための容器
。
メタルクラウン（ビット）
役割: 最先端に取り付けられ、地盤を直接削り取る刃先
。
💡 試験で問われやすい「連動ロジック」まとめ
掘進の3条件: 回転（変速装置）、荷重（スイベルヘッドの油圧）、**送水（ポンプ）**が適切に連動することで初めて効率的な掘進が可能です
。
水の流れ: 泥水バック → ポンプ → ウォータスイベル → ロッド内部（下降） → ビット先端 → ロッドと孔壁の間（上昇） → 泥水バック（循環）
。
動力の分岐: 一つの原動機からのパワーを、クラッチ操作によって「掘進（スイベルヘッド）」に使うか「昇降（巻上機）」に使うか、あるいは同時に使うかを切り替えます
。
この図2.5の各パーツを理解することは、機械の故障診断や事故防止（ロッド落下、逸水など）を理解するための基礎となります
。
